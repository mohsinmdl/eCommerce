How to Compare two commits on github


--------------------------------------------------------------------------
1- Go to the repository through which you want to compare changes
--------------------------------------------------------------------------

e.g
https://github.com/codingforentrepreneurs/eCommerce

--------------------------------------------------------------------------
2- Add "/compare" at the end of the repository
--------------------------------------------------------------------------

e.g
https://github.com/codingforentrepreneurs/eCommerce/compare

--------------------------------------------------------------------------------
3 - Add the commit no. in each of the two commit,brank fork search section
--------------------------------------------------------------------------------
